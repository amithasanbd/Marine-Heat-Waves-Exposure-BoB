# Data Folder

This folder is intentionally light for GitHub. Store large raw data outside the repository and archive processed/figure-ready data in Zenodo.

Suggested subfolders:
- `processed/`: cleaned processed data used by scripts.
- `figure_ready/`: minimal files required to recreate manuscript figures.
- `tables/`: CSV/XLSX tables supporting the manuscript and supplement.
- `external/`: notes and metadata for public source datasets.
