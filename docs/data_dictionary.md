# Data Dictionary

This file documents the recommended processed-data variables for the Zenodo dataset.

| Variable | Meaning | Suggested units |
|---|---|---|
| `sst` | Sea-surface temperature | degrees C |
| `mhw_event_id` | Unique marine heatwave event identifier | text |
| `start_date` | Event start date | YYYY-MM-DD |
| `end_date` | Event end date | YYYY-MM-DD |
| `duration_days` | MHW event duration | days |
| `mean_intensity` | Mean temperature above threshold during event | degrees C |
| `max_intensity` | Maximum temperature above threshold during event | degrees C |
| `cumulative_intensity` | Event-integrated thermal exposure | degrees C days |
| `mhw_days` | Number of MHW days per period | days |
| `event_frequency` | Number of events per period | count |
| `oni` | Oceanic Niño Index | degrees C |
| `dmi` | Dipole Mode Index | degrees C |
| `n2` | Brunt–Väisälä frequency squared | s^-2 |
| `mld` | Mixed-layer depth | m |
| `do` | Dissolved oxygen | mmol m^-3 |
| `no3` | Nitrate | mmol m^-3 |
| `po4` | Phosphate | mmol m^-3 |
| `si` | Silicate | mmol m^-3 |
